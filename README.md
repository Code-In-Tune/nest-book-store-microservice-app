# 📚 Book Store

## ❓ What it is

Book Store microservice in NestJS with Mongodb

## 🧬 Tech Stack

Technologies used:

- NestJS
- Class Validators
- Class Transformers
- Mongoose

## ✨ Features

The following features are being provided by the microservice:

- Add a book (POST)
- Update a book (PUT)
- Delete a book (DELETE)
- Search book by Id (GET)
- Get all books (GET)
- Add a book sale (POST)
- List all sales (GET)
- Get a book sale (GET)

## 🔥 Quick start

In order to start the project, first run the file docker-compose-mongo.yml.
Then run the application. In order to access the endpoints you can use the following endpoints with OpenAPI/Swagger:

- /docs/books
- /docs/sales
  The application runs on 3001 localhost.
